# T-Console-V2-Plus

> **Next-generation Open Source Infrastructure Operations Platform**
> **Status:** 🚧 Under Active Development

## Overview

T-Console-V2-Plus is a modern infrastructure operations platform designed to simplify enterprise IT management through a unified web interface.

The project aims to integrate infrastructure management, automation, monitoring, and future AI-assisted operations into a single extensible platform.

Current development focuses on building a stable foundation using modern technologies while preparing for future open-source collaboration.

## Planned Features

* IP Address Management (IPAM)
* Asset & Device Inventory
* Network Topology
* Microsoft 365 Integration
* Active Directory Integration
* Azure Integration
* VMware Integration
* Fortinet Integration
* F5 Integration
* Infrastructure Monitoring
* Automation Workflows
* REST API
* Role-Based Access Control (RBAC)
* Plugin Architecture
* AI Agent (Planned)

## Technology Stack

* FastAPI
* Vue 3
* PostgreSQL
* SQLAlchemy
* Tailwind CSS
* Docker

## Project Status

🚧 This project is currently under active development.

The architecture, features, and documentation are evolving rapidly. APIs, database schemas, and user interfaces may change before the first stable release.

## Vision

The long-term vision is to build an AI-powered Infrastructure Operations Platform capable of intelligent monitoring, infrastructure analysis, automated remediation, and workflow orchestration.

By making the project open source, we hope to provide enterprise IT administrators and infrastructure engineers with a modern, extensible alternative to traditional infrastructure management tools.
